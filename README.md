# Movie Notebook

This Express app is mainly a clone of GoodReads, but I wanted some slightly different functionality. Movie Notebook allows users to keep track of their favorite movies, as well as the favorite movies of directors and critics the user likes. This is my first project completely independent of App Academy.

## Technologies Used
App Academy largely focuses on the PERN stack. This project uses JavaScript, SQL, Sequelize, PostgreSQL, Express, Node.js, CSS, Pug, and Bcrypt.
